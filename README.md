MD2PDF
=====

[![platform](https://img.shields.io/badge/python-2.7-green.svg)]()

Convert markdown file to PDF(GitHub style).
功能简介：
* Markdown to HTML
* HTML to PDF

Markdown to PDF need wkhtmltopdf.
